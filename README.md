*Punteros, referencias
* Encapsulamiento/Herencia/Polimorfismo.

Yo elegí realizar una base de datos que pueda adecuarse a cualquier institucion que tenga alumnos y profesores.
Contiene las clases Persona, Estudiante, Profesor, Curso e Institucion
Se usa la libreria fstream para poder leer y escribir en un txt.


Las funciones que se podrán realizar son:
Ingresar nuevo alumno
Modificar datos de algun alumno
Buscar alumno por ID
Ver a los alumnos en el txt. ordenadas por apellido, alfabeticamente ascendente, descendente y por carreras.


Tuve algunas dificultades con las librerías para leer datos de archivos y en la asignacion de memoria dinamica, pues no sabía donde podía implementar eso.
